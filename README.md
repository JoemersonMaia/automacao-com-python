Automação com Python

Objetivo

O objetivo deste repositório é trazer projetos de automação utilizando Python, demonstrando minha vivência com a linguagem e podendo auxiliar pessoas que precisem implementar alguma funcionalidade parecida. 


Conteúdo atual

cadastro de produtos (finalizado):

O objetivo desta automação é criar um objeto que recebe uma planilha e executar tarefas manuais, como por exemplo automatizar o cadastro de produtos em um determninado sistema.
